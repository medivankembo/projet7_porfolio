Contexte du projet

Afin de pouvoir se vendre efficacement, les outils digitaux sont devenu un moyen indispensable pour présenter ses compétences et s'ouvrir aux opportunités.

Le but du projet est de développer un site internet Portfolio en suivant toutes partant de la réalisation des maquette jusqu'au développement du site internet.

Modalités pédagogiques

  • Le travail est individuel et est à réaliser en 10 jours

  • Le site doit être responsive

  • Le code doit être bien organisé en suivant le modèle de Airbnb dont le lien en ressource

  • Votre code doit CSS doit être écrit en SASS en respectant les conventions BEM

  • Un wireframe(Figma) de toutes les pages web doit être fourni en bonne et dû forme

  • Le contenu de votre site doit être dynamique et venir d'un fichier JSON crée en Fake REST API dont la documentation est mise comme ressource

​

​

Critères de performance

  • Vous devez faire attention aux choix des couleurs
  • Faire attention à la taille des images utilisé dans le site

Modalités d'évaluation

Une présentation et demonstration du projet devant le reste de la promotion

Livrables

Un repository GitHub contenant - La maquette de basse de fidélité(wireframe) - Le code source du projet - Un lien du site web hébergé dans un service autre que GitHub pages
